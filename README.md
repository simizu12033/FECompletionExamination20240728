# 基本情報技術者 科目A 全60問・図解解説

令和8年度（2026年）6月14日実施の修了認定試験を題材にした、静的な学習サイトです。

## ローカルで見る

このフォルダーで次を実行し、表示されたURLをブラウザーで開きます。

```bash
python -m http.server 8000
```

依存パッケージやビルド作業はありません。`index.html` を直接開いても基本機能は動きます。

## GitHub Pagesで公開する

1. このフォルダーの中身をGitHubリポジトリへ配置します。
2. リポジトリの `Settings` → `Pages` を開きます。
3. `Deploy from a branch` を選び、公開するブランチとフォルダーを指定します。

## ファイル構成

- `index.html` - ページ構造
- `styles.css` - レスポンシブデザイン
- `visuals.css` - 各問題専用の解説図
- `questions.js` - 全60問の解説データ
- `app.js` - 解説図の描画
- `learning.js` - 試験、採点、復習、理解チェックの進行管理
- `flow.css` - 試験・復習フローの表示
- `reference-map.html` - 参考問題対応表
- `reference-map.csv` - 参考問題対応表のCSV版
- `assets/questions/` - 提供された各問の問題文、選択肢、図表画像（Q01〜Q60）

問題画像は `q01.png`〜`q60.png` のPNGファイルだけを使用しています。

## 出典・参考

問題画像は提供された試験問題画像を使用しています。解説作成にあたり、
基本情報技術者試験、応用情報技術者試験、情報セキュリティマネジメント試験の
公開過去問題・関連問題も参考にしています。

各問の図解は、単純なフロー図だけでなく、対応図、比較図、階層図、計算式、
ネットワーク図、循環図など、学習内容に合った形式を使い分けています。

解答、採点段階、理解チェックはブラウザーの `localStorage` に自動保存されます。
初回採点時には、分野別正答率、得意・苦手分野、7日後の復習対象問題、
重要ワードを含む学習レポートも保存・表示されます。

保存データは同じ公開URL（オリジン）を同じブラウザーで開いた場合に保持されます。
別の端末・ブラウザー、シークレットモード、ブラウザーデータ削除後には引き継がれません。
